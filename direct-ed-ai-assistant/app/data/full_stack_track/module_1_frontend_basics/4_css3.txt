---

The most important thing to know about CSS3 is that it's **modular**. This allows the language to evolve more easily, with different modules being updated independently.

## 2. Key Features of CSS3
---

CSS3 introduced a wide range of powerful features that revolutionized web design, moving much of the work that previously required images or JavaScript directly into CSS.

### **Enhanced Selectors**
CSS3 added new selectors that allow for more precise targeting of HTML elements, giving developers finer control over styling without having to add extra classes or IDs to their HTML.

### **Advanced Borders and Backgrounds**
* **`border-radius`:** This simple property allows you to create rounded corners on elements, a design staple that previously required using images.
    `border-radius: 10px;`
* **`box-shadow`:** Adds a drop shadow effect around an element's frame.
    `box-shadow: 5px 5px 10px grey;`
* **Multiple Backgrounds:** You can now layer multiple background images on top of each other within a single element.

### **Gradients**
CSS3 introduced native support for creating color gradients, eliminating the need for image files to create smooth color transitions.
`background: linear-gradient(to right, blue, green);`

### **2D and 3D Transforms**
The `transform` property allows you to manipulate elements in 2D or 3D space. You can **rotate, scale, skew, and move (translate)** elements.
`transform: rotate(45deg) scale(1.5);`



### **Transitions and Animations**
These features brought motion design to the web without needing JavaScript or plugins.
* **Transitions:** Smoothly animate a property change from one state to another, for example, during a `:hover` event.
    `transition: background-color 0.5s ease-in-out;`
* **Animations (`@keyframes`):** Create complex, multi-step animations. You define the sequence of styles (keyframes) and then apply the animation to an element.

### **Responsive Design with Media Queries**
This is perhaps the most critical feature for the modern web. **Media Queries** allow you to apply different CSS rules based on the characteristics of the device, such as its screen width, height, or orientation. This is the foundation of **responsive web design**, which ensures that a website looks and functions well on desktops, tablets, and mobile phones.
```css
/* On screens that are 600px or less, make the body's background linen. */
@media screen and (max-width: 600px) {
  body {
    background-color: linen;
  }
}
